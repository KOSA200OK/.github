## 🙌 KOWROKS (200OK)

- 기획 동기 : 회사에는 공간, 차량, 인적자원 등 다양한 자원이 존재하는데, 이 자원이 효율적으로 운영되지 않는 경우가 많습니다. 회사 운영에서 자원 낭비를 최소화하기 위해 자원을 효율적으로 관리하는 회사자원 관리 시스템을 구축하고자 합니다.

- 차별점 : 기존에 있던 그룹웨어 기능에 더해 자원 관리 기능에 초점을 맞추어 개발하였습니다.
